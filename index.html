Chrome Installation Guide

1. Download NoFBRecommend.crx

Download

下載 NoFBRecommend.crx 檔案，由於並非由Chrome Web Store下載，因此會出現提示訊息，無法直接新增至瀏覽器內。

2. Goto chrome://extensions

於瀏覽器網址列輸入 chrome://extensions/ 進入瀏覽器擴充功能管理界面。

3. Drag NoFBRecommend.crx into Chrome

將 NoFBRecommend.crx 拖動進入瀏覽器即可開始安裝。

4. Add into Chrome

按下「新增」完成安裝。

5. Finish

完成安裝。

安裝後若未來釋出新版本，毋需至本站下載新版本替換，目前已經支援自動更新。

Safari Installation Guide

1. Download QCLean.safariextz ,then double click it.

Download

下載 QCLean.safariextz 檔案，並點擊開啓之。

2. Click Install

按下「安裝」完成安裝。

3. Finish

完成安裝。您可以在偏好設定中的延伸功能中找到它。

安裝後若未來釋出新版本，毋需至本站下載新版本替換，目前已經支援自動更新。
